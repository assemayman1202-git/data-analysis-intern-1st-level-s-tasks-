# Data Analysis Internship - Level 1 Tasks
This repository contains my work for the data analysis internship tasks, 
focusing on data cleaning, EDA, and visualization using Python.

## Dataset
- Boston Housing Dataset
- 506 rows × 14 columns
- Target variable: Median House Value (MEDV)


## Tasks
### Task 1: Data Cleaning & Preprocessing
- Added column headers
- Handled 54 missing values in Median Value column
- Fixed data types (CHAS → int)
- Checked for duplicates

### Task 2: Exploratory Data Analysis (EDA)
- Summary statistics
- Correlation heatmap
- Identified key features affecting house prices

### Task 3: Data Visualization
- Histograms for all features
- Boxplots for outlier detection
- Scatter plots

## Tools Used
- Python, Pandas, Matplotlib, Seaborn


## Key Findings
- Rooms (RM) has the strongest positive correlation with house price (0.74)
- Lower Status (LSTAT) has the strongest negative correlation (-0.71)
